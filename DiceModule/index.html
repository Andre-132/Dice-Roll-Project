Dice Roller

User Stories

Input Number of Sides: As a user, I want to input the number of sides for the dice, so I can customize the dice roll. The Dice: As a user, I want to press a button to roll the dice, so I can see the result of the roll based on my input. Display Result: As a user, I want to see the dice roll result on the screen, so I can know the result. Error: As a user, I want to be notified if my input is invalid so I can correct it.

Storyboard

Load the Page, The user opens the web page and sees an input field labeled "Number of Sides," a "Roll Dice" button, and a result section.Enter Input, The user types a number into the input field to specify the number of sides for the dice.Roll the Dice, The user clicks the "Roll Dice" button. The system generates a random number between 1 and the specified number of sides.Display the Result, The result "You is displayed on the screen, replacing the previous empty result section.Invalid Input If the user enters invalid input, an error message appears prompting them to correct it.
